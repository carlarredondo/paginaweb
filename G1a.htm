PÁGNAS WEB

Para construir páginas dinámicas se utilizan varios lenguajes y tecnologías:

  • HTML: sigue siendo la base de la estructura de la página que se enviará al navegador.

  • ASP (Active Server Pages): un lenguaje de programación del lado del servidor desarrollado por Microsoft que permite generar contenido dinámico en páginas web y conectarse a bases de datos.

  • PHP (Hypertext Preprocessor, en tu mensaje lo escribiste como THP): un lenguaje de servidor muy popular para crear páginas dinámicas que interactúan con bases de datos SQL.

  • Java y JSP (Java Server Pages): Java se utiliza para la programación del servidor y JSP permite insertar código Java dentro del HTML para generar páginas dinámicas de forma más organizada.

  • SQL (Structured Query Language): es el lenguaje que permite interactuar con bases de datos, como MySQL, PostgreSQL o SQL Server, para almacenar, consultar, modificar y eliminar información. Las páginas dinámicas utilizan SQL para mostrar contenido que depende de los datos guardados, como productos, usuarios o publicaciones.

Construcción de una página dinámica:

-          El usuario solicita una página desde el navegador.

-          El servidor recibe la solicitud y ejecuta código de ASP, PHP, Java o JSP.

-          Ese código consulta la base de datos mediante SQL para obtener la información necesaria.

-          El servidor genera HTML dinámico con los datos obtenidos y lo envía al navegador.

-          El navegador muestra la página ya “armada”, que puede variar para cada usuario o situación.
