PÁGNAS WEB

 Para construir una página estática se utilizan principalmente:

  • HTML (HyperText Markup Language): es el lenguaje principal que define la estructura de la página. A través de etiquetas HTML se indican títulos, párrafos, listas, tablas, imágenes, enlaces y otros elementos. Por ejemplo, un encabezado <h1> se utiliza para mostrar el título principal, mientras que un párrafo <p> contiene texto normal.
  • CSS (Cascading Style Sheets): se encarga del diseño y la apariencia visual de la página. Con CSS se definen colores, fuentes, márgenes, tamaños, alineaciones y cualquier estilo que haga que la página sea visualmente atractiva. Por ejemplo, podemos hacer que los encabezados sean de color azul y centrados con unas pocas líneas de código CSS.
  • JavaScript (opcional): permite agregar interactividad a la página, aunque sin conexión a bases de datos o lógica compleja del servidor. Con JavaScript se pueden crear menús desplegables, validación de formularios, animaciones, efectos visuales y pequeñas interacciones con el usuario.

Construcción de una página estática:

-          Se crea el archivo HTML que contiene la estructura del contenido.

-          Se añade un archivo CSS para definir el diseño y la presentación visual de los elementos.

-          Se puede incluir un archivo JavaScript para agregar interactividad limitada.

Por ejemplo, un sitio web estático sencillo podría consistir en un archivo HTML que muestre un título y un párrafo, mientras que el CSS define colores y fuentes, y el JavaScript agrega un pequeño efecto al pulsar un botón. Todas estas páginas se sirven tal cual desde el servidor y no requieren procesamiento adicional ni conexión con bases de datos.
